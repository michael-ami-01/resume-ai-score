# ResumeAI Score — Free AI Resume Grader

Get your resume scored by AI instantly. 6-dimension analysis with actionable feedback on impact, clarity, keywords, ATS compatibility, and more.

🔗 **[Try it live →](https://michael-ami-01.github.io/resume-ai-score/)**

## Features

- **6 scoring dimensions**: Impact & Results, Clarity & Structure, Keywords & ATS, Experience, Technical Strength, Polish
- **Detailed feedback** with severity levels (Critical / Warning / Tip)
- **Strengths & improvements** summary
- **Optional target role** for tailored analysis
- **Career niche**: The highest-CPC niche in AI tools ($3-5 CPC)

## Usage

Bring your own API key (Anthropic Claude or OpenAI). Your resume stays in your browser.
